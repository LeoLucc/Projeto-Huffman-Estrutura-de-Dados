node_t * montar_arvore ( FILE * archive,
short int * treeSize )

Monta a árvore de Huffman a partir do arquivo compactado.

Esta função lê os dados do arquivo compactado em pré-ordem e monta a árvore de Huffman de acordo com o tamanho fornecido.

Parâmetros
archivePonteiro para o arquivo compactado.
treeSizePonteiro para um inteiro que armazena o tamanho da árvore.
Retorna
Retorna um ponteiro para a árvore de Huffman montada.

◆ tamanho_arvore_arquivoCompactado()

int tamanho_arvore_arquivoCompactado ( FILE * archive)

Obtém o tamanho da árvore do arquivo compactado.

Esta função lê os dois primeiros bytes do arquivo compactado e extrai os 5 bits menos significativos do primeiro byte e os 8 bits do segundo byte para determinar o tamanho da árvore.

Parâmetros
archivePonteiro para o arquivo compactado.
Retorna
Retorna o tamanho da árvore em bits.

int tamanho_lixo_arquivoCompactado ( FILE * archive)

Obtém o tamanho do lixo do arquivo compactado.

Esta função lê o primeiro byte do arquivo compactado e extrai os 3 bits mais significativos para determinar o tamanho do lixo.

Parâmetros
archivePonteiro para o arquivo compactado.
Retorna
Retorna o tamanho do lixo em bits, ou -1 se o arquivo não puder ser lido.
