◆ arvoreHuff()

node_t * arvoreHuff ( node_t ** head,
int * tamLista )

Monta a árvore de Huffman a partir de uma lista ordenada.

Esta função constrói a árvore de Huffman com base nos nós fornecidos em uma lista ordenada. Os dois nós de menor frequência são removidos da lista e um novo nó pai é criado. O processo continua até que reste apenas um nó, que se torna a raiz da árvore.

Parâmetros
headPonteiro para o ponteiro da lista de nós a serem utilizados.
tamListaPonteiro para um inteiro que irá armazenar o tamanho da lista.
Retorna
Retorna o ponteiro para o nó raiz da árvore de Huffman.

bool ehFolha ( node_t ** head)

Verifica se um nó é uma folha da árvore.

Esta função verifica se o nó fornecido é uma folha, ou seja, se não possui filhos esquerdo ou direito.

Parâmetros
headPonteiro para o ponteiro da raiz da árvore.
Retorna
Retorna true se o nó for uma folha, caso contrário, retorna false.

Calcula o tamanho da árvore.

Esta função calcula e retorna o número de nós presentes na árvore, considerando um ajuste especial para nós folha que têm caracteres específicos ('*' e '\').

Parâmetros
noPonteiro para o nó que serve como raiz da árvore.
Retorna
